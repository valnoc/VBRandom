VBRandom
========
[![](https://img.shields.io/cocoapods/v/VBRandom.svg)]()

VBRandom is a pseudo-random numbers generator. Using of VBRandom simplifies random numbers creation code.

It uses arc4random() function.

## Supported features
1. random NSUInteger
2. random NSInteger
3. random double
4. random BOOL
